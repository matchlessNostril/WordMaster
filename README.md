## 💻 Introduction

Have you ever needed to memorize **both the spelling and pronunciation** of words?

Did you buy a vocabulary book but find it difficult to repeatedly **test yourself conveniently**?

With **Word Master**, you can create **your own vocabulary lists**,  
select the word sets you want to test,  
and take tests **anytime, as much as you want, freely and conveniently!**

### 4️⃣ Create Word Set

<img src="https://github.com/user-attachments/assets/8dfdcf32-70f3-4bf7-9cca-9b28a59978aa" width="900" />

- Enter words into your categorized **word sets**.
- **Words, meanings, and pronunciation fields** are available.
  - If you're studying **Japanese**, pronunciation is crucial because of **kanji**! 🫠
  - However, **testing only pronunciation** is difficult with traditional methods.
  - But with **Word Master**, pronunciation can be included in tests!
- **Delete words** with the trash icon.
- **Add more words** with the plus button.
- Click **Create** when finished to save the set and move to the word set screen.
